##### Usage
1. Home Page: Main portfolio showcasing background, experience, as well as skills.
2. Gallery Page: Page to showcase photography as well as different gallery implementations
3. FAQ Page: To provide information to users for common issues within the application that I currently support. This content has been included to showcase my skills with managing and organizing content within a site.
4. Products Page: Showcase for past, current, and upcoming projects in the works.
5. Denim Page: Personal project to provide content to work with and manage. This is also a fun way to show others one of my many interests.
